Cafetería Escolar en Solana

Este proyecto consiste en el desarrollo de un programa en la blockchain de Solana, utilizando Rust y el framework Anchor Framework dentro del entorno de Solana Playground.

El proyecto originalmente estaba basado en un sistema de biblioteca, pero fue adaptado a una cafetería escolar, donde se implementa un CRUD básico para administrar productos del menú. A través del programa es posible:

Crear una cafetería.

Agregar productos al menú.

Eliminar productos.

Visualizar la lista de productos disponibles.

Cambiar la disponibilidad de los productos.

Todas estas operaciones se almacenan y ejecutan directamente en la blockchain, permitiendo practicar el desarrollo de programas descentralizados (Smart Contracts) en Solana.

Este proyecto fue realizado como parte de las prácticas necesarias para comprender el funcionamiento del desarrollo en Solana y obtener la certificación oficial de Solana.
